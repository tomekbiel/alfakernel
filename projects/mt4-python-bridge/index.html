System Overview

The MT4 Trading System is a sophisticated trading infrastructure that connects Python's analytical capabilities with MetaTrader 4's execution environment. This bridge allows for advanced algorithmic trading strategies, data analysis, and automated trade execution.

Key Components

1. MT4 Connector (mt4_connector/)

The core bridge between Python and MT4, handling all communication:

Base Functionality (base_connector.py):

  • TCP/IP communication with MT4 Expert Advisor

2. Trading Strategies

Example strategy implementation:

def moving_average_cross(symbol, timeframe='M15', short_period=9, long_period=21):
    """
    Simple moving average crossover strategy.
    Returns: 'BUY', 'SELL', or None
    """
    # Get historical data
    data = connector.get_historical_data(symbol, timeframe, bars=100)
    
    # Calculate indicators
    data['SMA_short'] = data['close'].rolling(window=short_period).mean()
    data['SMA_long'] = data['close'].rolling(window=long_period).mean()
    
    # Generate signals
    if data['SMA_short'].iloc[-1] > data['SMA_long'].iloc[-1] and \
       data['SMA_short'].iloc[-2] <= data['SMA_long'].iloc[-2]:
        return 'BUY'
    elif data['SMA_short'].iloc[-1] < data['SMA_long'].iloc[-1] and \
         data['SMA_short'].iloc[-2] >= data['SMA_long'].iloc[-2]:
        return 'SELL'
    return None

Getting Started

1. Installation

  • Download: Get the latest release from GitHub.
  • MT4 Setup: Download DWX_ZeroMQ_Server_v2.0.1_RC8.ex4 and copy it to your MetaTrader 4 MQL4/Experts folder, then compile.
  • Python Setup: Install requirements:
    pip install -r requirements.txt
  • Run Bridge: Start the Expert Advisor in MT4 and run mt4_bridge.py in Python.
  • Configuration: Adjust config.py and MT4 expert settings as needed.
  • For detailed instructions, see the README on GitHub.

2. Basic Usage

from mt4_connector import MT4CommandSender

# Initialize connection
connector = MT4CommandSender(host="127.0.0.1", port=8888)
