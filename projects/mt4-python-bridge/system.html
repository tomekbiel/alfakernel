Architecture

The bridge connects MetaTrader 4 (via a custom Expert Advisor using ZeroMQ) to a Python backend that handles data retrieval, analysis, and automation.

  • 📡 MT4 Expert listens on ports 5555–5557 (REQ/REP, PUSH/PULL, PUB/SUB)
  • 🐍 Python includes modular handlers for:
    • base_connector.py – Core messaging logic
    • market_data_handler.py – Live data via SUB
    • historical_data_handler.py – History fetch via COMMAND
    • trade_handler.py – Send trade commands
  • 📂 Scripts automate common tasks like:
    • fetch_single.py – Download historical data
    • run_mt4.py – Test connector and strategy interface
