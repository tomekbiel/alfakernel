1. MetaTrader 4 Setup

You need a working MetaTrader 4 terminal with MQL4 support. Follow these steps:

  1. Open MT4, go to File ‚Üí Open Data Folder
  2. Copy the file DWX_ZeroMQ_Server_v2.0.1_RC8.ex4 from the files/ folder into:
    MQL4/Experts/
  3. Copy all files from files/library/ into:
    MQL4/Libraries/
  4. Restart MT4
  5. Launch MetaEditor (from the MT4 toolbar), open the EA, and compile it if needed
  6. Drag the expert DWX_ZeroMQ_Server_v2.0.1_RC8 onto any chart (e.g., EURUSD M1)
  7. You should see a smiley face üòä in the top-right corner of the chart

2. Python Environment

This project uses Python 3.9+ (Anaconda is recommended). Required packages: 

pip install pyzmq pandas numpy

Clone or download the Python scripts from the GitHub repository:

github.com/tomekbiel/MT4_Trading_System

Use the provided launcher (e.g. run_mt4.py) to test communication.

3. Communication Test

  1. Start MT4 and ensure the expert advisor is running (smiley face visible)
  2. Run the Python launcher script
  3. Observe terminal messages ‚Äì if you see account info or prices, the bridge works!
