R Data Analysis

This section contains a collection of R scripts, reports, and interactive analyses.
Select a topic from the sidebar to view details.

Analysis of Variance (ANOVA)

Testing the significance of differences between means in two or more groups. Examines if a grouping variable affects an outcome variable.

Multiple Regression

Modeling the relationship between one dependent (outcome) variable and two or more independent (explanatory) variables. Prediction and assessment of predictor influence strength.

Nonparametric Regression Methods

Regression techniques that do not require the assumption of normally distributed data. Robust to violations of classical regression assumptions.

Discriminant Analysis

Classifying observations into predefined groups based on multiple explanatory variables. Finding combinations of variables that best separate the groups.

Classification and Regression Trees (CART - Basic)

Building interpretable tree-based models for prediction (classification or regression) through recursive partitioning of data.

Classification and Regression Trees - Multi-Model Approach

Ensembles of trees (e.g., Random Forests, Boosting) that improve prediction accuracy and stability compared to a single tree.

Conjoint Analysis

Studying consumer preferences. Estimating the relative importance (utility) people assign to different attributes of a product/service based on their stated choices.

Factor Analysis (FA)

Data dimensionality reduction. Identifying underlying, unobservable "factors" that explain correlations among many observed variables.

Multidimensional Scaling (MDS)

Visualization of similarity or dissimilarity between objects in a lower-dimensional space (usually 2D or 3D).

Correspondence Analysis (CA)

Visualization and exploration of associations between categorical variables in a two-way contingency table. Analogous to PCA for qualitative data.

Cluster Analysis (Clustering)

Discovering natural groups ("clusters") within data based on similarity between observations. No prior group labels are known.

Cluster Analysis - Model-Based Approach

Clustering methods based on statistical distributional assumptions of the data (e.g., Gaussian Mixture Models - GMM). Determining the optimal number of clusters.
